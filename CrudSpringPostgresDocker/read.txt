1.add dependencies to pom.xml file
<dependency>
<groupId>org.postgresql</groupId>
<artifactId>postgresql</artifactId>
<scope>runtime</scope>
</dependency>
<dependency>
<groupId>org.springframework.boot</groupId>
<artifactId>spring-boot-starter-web</artifactId>
</dependency>

2. Add docker-compose.yaml  file
3. Add Docker file
4. Ensure following structure https://github.com/GozdeSaygiliYalcin/docker-compose-demo
my-docker-project/
   |-- src/
   |   |-- main/
   |        |-- java/ [source root]
   |-- docker-compose.yml
   |-- Dockerfile

5. mvn package -DskipTests
6. docker build . -t spring-postgres
7. docker-compose up //docker-compose down
8. http://localhost:8081/customers
